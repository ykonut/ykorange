> Mac 文件系统对快速索引文件的支持天然弱于 Windows，可能时常会碰到 Raycast 搜不到文件的情况。目前 Mac 上效果最好的可能是 OnyX 基于 `mdfind` 命令构建的搜索功能。`mdfind` 使用与 Spotlight 相同的元数据，不过实测比直接使用 Spotlight 效果更好；你也可以直接在命令行使用 `mdfind -name` 进行搜索。

### Markdown

- [Obsidian](https://obsidian.md/)
- 插件
	- `Remotely Save`：为笔记库设置云同步
	- `Easy Typing`：中文输入增强
	- `Latex Suite`：数学公式输入增强
	- `Recent Files`：显示最近文件列表
	- `Enhancing Export`：基于 [Pandoc](https://pandoc.org/) 扩展导出格式（你甚至可以用来做 PPT）
	- `Digital Garden`：使用笔记建立网站

### 电子书管理

- [Readest](https://readest.com/)：电子书阅读软件。
- [Calibre](https://calibre-ebook.com/zh_CN)：最好（几乎唯一）的电子书管理软件。参考[教程](https://sspai.com/post/72748)。
- 插件：
	- [Z-Library Calibre Store](https://github.com/tekofx/zlibrary-calibre-store-plugin)：直接在 Calibre 中检索 Z-library 图书
	- [NewDouban](https://github.com/fugary/calibre-douban)：从豆瓣获取元数据
	- [国家图书馆ISBN](https://github.com/DoiiarX/NLCISBNPlugin)：基于 ISBN 从国家图书馆获取元数据
	- [No Trans](https://github.com/Cirn09/calibre-do-not-translate-my-path)：禁止 Calibre 将中文路径转换为拼音
- [bilingual_book_maker](https://github.com/yihong0618/bilingual_book_maker) 🤖：制作双语 epub 电子书。推荐使用 `uv tool install bbook-maker` 安装。
