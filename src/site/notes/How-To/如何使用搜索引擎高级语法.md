## 1. 精确匹配关键词 `"xxx"`

|              需求               |             搜索示例              |
| :---------------------------: | :---------------------------: |
|      查找含有完整短语“机器学习入门”的页面      |          `"机器学习入门"`           |
| 查找包含原句“to be or not to be”的文档 |    `"to be or not to be"`     |
|           精确搜索某论文题目           | `"Attention Is All You Need"` |

## 2. 限定标题包含关键词 `intitle:`

|           需求            |           搜索示例           |
| :---------------------: | :----------------------: |
|    查找标题中包含“深度学习”的文章     |      `intitle:深度学习`      |
|     找 Python 教程类页面      |   `intitle:Python 教程`    |
| 查找标题含“Transformer”的论文解读 | `intitle:Transformer 论文` |

## 3. 限定标题多个关键词 `allintitle:`

|            需求            |            搜索示例             |
| :----------------------: | :-------------------------: |
|    标题同时含“R 语言”和“数据分析”     |    `allintitle:R语言 数据分析`    |
| 查找含“Transformer”和“讲解”的页面 | `allintitle:Transformer 讲解` |
|    查含“Polars”和“性能”的文章    |   `allintitle:Polars 性能`    |

## 4. 限定正文内容关键词 `intext:`

|             需求              |             搜索示例              |
| :-------------------------: | :---------------------------: |
|      查找正文包含“协方差矩阵”的讲义       |        `intext:协方差矩阵`         |
| 查正文含“bootstrap sampling”的资料 | `intext:"bootstrap sampling"` |
|   查正文包含“Markov chain”的教程    |    `intext:"Markov chain"`    |

## 5. 限定网址关键词 `inurl:`

|              需求               |          搜索示例           |
| :---------------------------: | :---------------------: |
| 查 URL 中带 tutorial 的 Python 教程 | `inurl:tutorial Python` |
|     查 URL 中含 pdf 的深度学习资料      |    `inurl:pdf 深度学习`     |
|      查含 dataset 路径的数据集页面      |   `inurl:dataset NLP`   |

## 6. 限定网站来源 `site:`

|          需求           |               搜索示例                |
| :-------------------: | :-------------------------------: |
|      在政府网站搜索“公示”      |         `site:gov.cn 公示`          |
|      在教育网站搜索“张三”      |         `site:edu.cn 张三`          |
| 在 GitHub 搜索 Polars 教程 | `site:github.com Polars tutorial` |

## 7. 限制图片尺寸 `imagesize:`

|             需求             |                   搜索示例                   |
| :------------------------: | :--------------------------------------: |
|         查找 4K 风景壁纸         | `mountain landscape imagesize:3840x2160` |
| 查 5760×3840 的 Elon Musk 照片 |     `Elon Musk imagesize:5760x3840`      |
|     查 2560×1600 的宇宙壁纸      |  `galaxy wallpaper imagesize:2560x1600`  |

|   标准    |    分辨率    | 常用别名  |
| :-----: | :-------: | :---: |
|   HD    | 1280×720  | 720p  |
| Full HD | 1920×1080 | 1080p |
|   QHD   | 2560×1440 |  2K   |
|  WQHD   | 2560×1600 | 16:10 |
| 4K UHD  | 3840×2160 |  4K   |

## 8. 限制文件格式 `filetype:`

|       需求        |          搜索示例          |
| :-------------: | :--------------------: |
| 查 PPT 形式的深度学习课件 |  `深度学习 filetype:ppt`   |
|  查 Polars 教程文档  | `Polars filetype:pdf`  |
| 查 R 语言教材 Word 版 | `R语言 教程 filetype:docx` |
